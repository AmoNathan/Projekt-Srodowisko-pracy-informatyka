# ProjektZaliczeniowy Srodowisko Pracy Informatyka
Projekt dotyczy aplikacji mobilnej "Tinder dla samochodow" o nazwie CAR MATCH.
W aplikacji bedzie mozna ocenia auta innych uzytkownikow oraz dodawac swoje samochody.
Odbiorcami beda fani motoryzacji, oraz firmy zamujace sie handlem samochodami dla mozliwosci promocji swojej firmy.
